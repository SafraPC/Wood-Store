O Que é a Wood Store?

FICÇÃO:

 A Wood Store é uma Start up Criada em 2020 pelo empresário de sucesso Leandro Safra, que tem como objetivo vender móveis planjeados, itens de decoração e as madeiras mais cheirosas e bem produzidas de toda a região de Campinas!
Com suas habilidades em HTML5 e CSS3 ele mostrará que é capaz de criar carrosséis, um design atrativo e um código limpo para que os usuários possam ter as melhores experiências e uma interface super limpa e de fácil acesso.

REALIDADE:

 A Wood Store é um projeto simples de um e-commerce sem framework algum para mostrar as minhas mais puras habilidades em HTML5, CSS3 e JavaScript. com este projeto, tenho a intenção de colocar o básico para que qualquer usuário tenha uma experiência fácil e de interface limpa, ou seja, focando totalmente em UI/UX e trabalhando com código limpo e uma arquitetura básica que qualquer desenvolvedor possa entender.
ATENÇÃO: ESTE É APENAS UM PROJETO DE PORTIFÓLIO!
